========================================
БЕЗОПАСНОСТЬ - WINDOWS И MINT НА ОДНОМ ДИСКЕ
========================================

ВОПРОС: Не повредит ли восстановление GRUB мой Windows?
ОТВЕТ: НЕТ! Это полностью безопасно.

ПОЧЕМУ БЕЗОПАСНО:
-----------------
1. GRUB устанавливается только в загрузочную область диска
2. Windows раздел (C:\) НЕ затрагивается
3. Windows файлы остаются нетронутыми
4. GRUB просто добавляет меню выбора при загрузке

ЧТО ПРОИСХОДИТ:
---------------
ДО восстановления:
  Компьютер → Windows (сразу)

ПОСЛЕ восстановления:
  Компьютер → GRUB меню → Выбор:
    - Linux Mint
    - Windows Boot Manager

Windows работает точно так же, просто появляется меню выбора!

ЕСЛИ ЧТО-ТО ПОЙДЕТ НЕ ТАК:
--------------------------
1. Windows можно загрузить через BIOS/UEFI меню (F12)
2. Windows можно восстановить через Windows Recovery
3. Можно вернуть Windows загрузчик командой:
   bootrec /fixmbr
   bootrec /fixboot
   bootrec /rebuildbcd

НО ЭТО НЕ ПОНАДОБИТСЯ - GRUB безопасен!

СТРУКТУРА ДИСКА (пример):
--------------------------
Физический диск: /dev/sda

├── /dev/sda1  EFI раздел (загрузчик) ← GRUB установится сюда
├── /dev/sda2  Linux Mint (ваши файлы Mint)
├── /dev/sda3  Linux swap (опционально)
└── /dev/sda4  Windows C:\ (ваши файлы Windows) ← НЕ ЗАТРАГИВАЕТСЯ!

ВЫВОД:
------
✓ Восстановление GRUB безопасно
✓ Windows не пострадает
✓ Просто появится меню выбора
✓ Можно вернуть как было (если нужно)
