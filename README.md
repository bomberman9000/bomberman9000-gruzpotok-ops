# 🚀 Оптимизация Windows

Набор PowerShell скриптов для оптимизации и настройки Windows 10/11.

## 📋 Содержание

- **optimize-windows.ps1** - Главный скрипт, запускающий все оптимизации
- **clean-system.ps1** - Очистка временных файлов и кэша
- **performance-tweaks.ps1** - Оптимизация производительности
- **security-tweaks.ps1** - Настройка безопасности

## ⚠️ Важно!

**Все скрипты должны запускаться от имени администратора!**

1. Нажмите правой кнопкой мыши на файл `.ps1`
2. Выберите "Запустить с PowerShell" или "Запустить от имени администратора"

## 🚀 Быстрый старт

### Полная оптимизация + красивое оформление (рекомендуется)
```powershell
.\optimize-windows.ps1 -All
```

Это выполнит:
- Очистку системы
- Оптимизацию производительности
- Настройку безопасности
- **Красивое оформление Windows** ✨

### Отдельные модули

**Только очистка:**
```powershell
.\optimize-windows.ps1 -Clean
```

**Только производительность:**
```powershell
.\optimize-windows.ps1 -Performance
```

**Только безопасность:**
```powershell
.\optimize-windows.ps1 -Security
```

**Только красивое оформление:**
```powershell
.\optimize-windows.ps1 -Beauty
```

## 📝 Что делают скрипты

### 🎨 beauty-tweaks.ps1 (НОВОЕ!)
- Включает темную тему
- Настраивает красивые цвета и прозрачность
- Оптимизирует панель задач и меню Пуск
- Улучшает проводник
- Отключает рекламу
- Включает визуальные эффекты

### 🧹 clean-system.ps1
- Очищает временные файлы
- Очищает корзину
- Очищает кэш DNS
- Очищает кэш Windows Store
- Очищает логи Windows
- Очищает кэш браузеров

### ⚡ performance-tweaks.ps1
- Оптимизирует визуальные эффекты
- Настраивает службы Windows
- Оптимизирует реестр
- Настраивает план питания на максимальную производительность
- Отключает телеметрию
- Оптимизирует настройки для SSD
- Отключает дефрагментацию для SSD

### 🔒 security-tweaks.ps1
- Включает и настраивает брандмауэр
- Настраивает UAC
- Отключает автозапуск с USB/CD
- Отключает устаревший SMBv1
- Настраивает Windows Update
- Настраивает защиту от эксплойтов
- Настраивает SmartScreen

## 🔄 После выполнения

После выполнения скриптов рекомендуется:
1. **Перезагрузить компьютер** для применения всех изменений
2. Проверить, что все работает корректно
3. При необходимости откатить изменения через точку восстановления

## ⚙️ Создание точки восстановления (рекомендуется)

Перед запуском скриптов создайте точку восстановления:

```powershell
Checkpoint-Computer -Description "Перед оптимизацией Windows" -RestorePointType "MODIFY_SETTINGS"
```

## 🔙 Откат изменений

Если что-то пошло не так, используйте точку восстановления:
1. Откройте "Панель управления" → "Восстановление"
2. Выберите "Запуск восстановления системы"
3. Выберите точку восстановления, созданную перед оптимизацией

## 📌 Примечания

- Некоторые изменения требуют перезагрузки
- Скрипты безопасны, но всегда рекомендуется создавать точку восстановления
- Некоторые службы могут быть отключены - при необходимости их можно включить обратно
- Настройки можно изменить вручную через реестр или групповые политики

## 🛠️ Требования

- Windows 10/11
- PowerShell 5.1 или выше
- Права администратора

## ⚠️ Отказ от ответственности

Используйте скрипты на свой страх и риск. Автор не несет ответственности за возможные проблемы. Всегда создавайте точку восстановления перед использованием!

## 📞 Поддержка

При возникновении проблем:
1. Проверьте, запущены ли скрипты от имени администратора
2. Проверьте логи PowerShell на наличие ошибок
3. Используйте точку восстановления для отката изменений
