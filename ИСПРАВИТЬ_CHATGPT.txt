========================================
ИСПРАВИТЬ ДОСТУП К CHATGPT
========================================

ПРОБЛЕМА: Не могу зайти в ChatGPT уже час

РЕШЕНИЕ 1: БЫСТРОЕ (БЕЗ ПРОКСИ ДЛЯ CHATGPT)
--------------------------------------------

1. Откройте Firefox
2. Меню (☰) → Настройки
3. Прокрутите до "Сетевые параметры"
4. Нажмите "Параметры..."
5. В поле "Не использовать прокси для:" добавьте:
   *.openai.com, *.chatgpt.com, *.anthropic.com
6. Нажмите OK
7. Перезапустите Firefox
8. Попробуйте открыть: https://chat.openai.com

РЕШЕНИЕ 2: ОТКЛЮЧИТЬ ПРОКСИ ВРЕМЕННО
-------------------------------------

1. Откройте Firefox
2. Меню (☰) → Настройки → Сетевые параметры → Параметры...
3. Выберите "Без прокси"
4. Нажмите OK
5. Попробуйте открыть ChatGPT
6. После работы включите прокси обратно

РЕШЕНИЕ 3: ИСПОЛЬЗОВАТЬ ДРУГОЙ БРАУЗЕР
---------------------------------------

Chrome/Edge используют системные настройки прокси:
1. Откройте Chrome или Edge
2. Попробуйте зайти в ChatGPT
3. Если не работает, отключите системный прокси:
   - Параметры Windows → Сеть и Интернет → Прокси
   - Отключите "Использовать прокси-сервер"

РЕШЕНИЕ 4: АВТОМАТИЧЕСКОЕ
--------------------------

Запустите скрипт:
.\fix-chatgpt.ps1

Он настроит исключения для ChatGPT автоматически.

ПОЧЕМУ НЕ РАБОТАЕТ:
-------------------

1. ChatGPT может блокировать VPN/прокси
2. Прокси может быть перегружен
3. DNS через прокси может работать медленно
4. Нужны исключения для доменов OpenAI

ПРОВЕРКА:
---------

1. Проверьте Hysteria2:
   Get-Process -Name "hysteria2"

2. Проверьте порты:
   netstat -an | findstr "8080 1080"

3. Проверьте доступность ChatGPT:
   - Откройте: https://chat.openai.com
   - Или: https://www.chatgpt.com

4. Проверьте в режиме инкогнито:
   - Ctrl+Shift+P в Firefox
   - Попробуйте открыть ChatGPT

ЕСЛИ ВСЕ ЕЩЕ НЕ РАБОТАЕТ:
-------------------------

1. Отключите прокси полностью
2. Попробуйте другой VPN/прокси
3. Проверьте, не заблокирован ли ChatGPT в вашей стране
4. Попробуйте через мобильный интернет (если есть)

БЫСТРАЯ КОМАНДА:
----------------

Отключить прокси в Firefox через about:config:
1. Откройте: about:config
2. Найдите: network.proxy.type
3. Установите: 0 (без прокси)
4. Перезапустите Firefox
