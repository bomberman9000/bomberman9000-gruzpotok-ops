# 🎨 Красивое оформление Windows

Дополнительные скрипты для красивого оформления Windows.

## 📋 Новые возможности

### ✨ beauty-tweaks.ps1
Скрипт делает Windows красивой и современной:

**Что настраивается:**
- 🌙 **Темная тема** - включается для всех приложений
- 🎨 **Цвета акцента** - настраивается красивый синий цвет
- ✨ **Прозрачность** - включаются эффекты прозрачности
- 📊 **Панель задач** - оптимизируется и очищается
- 🚀 **Меню Пуск** - настраивается для удобства
- 📁 **Проводник** - улучшается интерфейс
- 🔕 **Реклама** - отключается вся реклама в системе
- 🎭 **Визуальные эффекты** - включаются красивые анимации
- 🖼️ **Обои** - настраивается позиционирование

### 🖼️ wallpapers.ps1
Скрипт для установки красивых обоев:

1. Создайте папку `wallpapers` в папке проекта
2. Добавьте туда свои обои (jpg, png, bmp)
3. Запустите скрипт - он установит обои автоматически

## 🚀 Как использовать

### Полная оптимизация + красота:
```powershell
.\optimize-windows.ps1 -All
```

### Только красивое оформление:
```powershell
.\optimize-windows.ps1 -Beauty
```

### Установка обоев:
```powershell
.\wallpapers.ps1
```

## 🎯 Что изменится

После запуска `beauty-tweaks.ps1`:

✅ **Темная тема** включена везде
✅ **Прозрачность** и эффекты работают
✅ **Панель задач** чистая и оптимизированная
✅ **Меню Пуск** без рекламы
✅ **Проводник** показывает расширения файлов
✅ **Визуальные эффекты** включены
✅ **Реклама** полностью отключена

## 💡 Дополнительные советы

### Установка своих обоев:
1. Найдите красивые обои в интернете
2. Сохраните в папку `wallpapers`
3. Запустите `wallpapers.ps1`

### Настройка цветов:
- Откройте: Параметры → Персонализация → Цвета
- Выберите свой цвет акцента
- Включите "Показывать цвет акцента в меню Пуск, на панели задач и в центре уведомлений"

### Дополнительные темы:
- Windows Store → Темы
- Скачайте понравившуюся тему
- Примените её

## ⚠️ Важно

- Скрипты требуют прав администратора
- После выполнения рекомендуется перезагрузка
- Некоторые изменения применяются сразу
- Проводник перезапускается автоматически
