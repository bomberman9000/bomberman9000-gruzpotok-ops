========================================
  ИСПРАВЛЕНИЕ MICROSOFT STORE
========================================

ПРОБЛЕМА: Microsoft Store не работает из-за ограничений

РЕШЕНИЕ: Используйте скрипт fix-store.ps1

========================================
  КАК ЗАПУСТИТЬ:
========================================

1. Откройте PowerShell ОТ ИМЕНИ АДМИНИСТРАТОРА
   
   Способ 1:
   - Нажмите Win + X
   - Выберите "Windows PowerShell (администратор)"
   
   Способ 2:
   - Нажмите Win + R
   - Введите: powershell
   - Нажмите Ctrl + Shift + Enter

2. Выполните команды:

   cd C:\Users\Shata\project
   .\fix-store.ps1

========================================
  ЧТО ДЕЛАЕТ СКРИПТ:
========================================

Шаг 1: Сбрасывает приложение Store
Шаг 2: Удаляет ограничения из реестра
Шаг 3: Включает необходимые службы
Шаг 4: Перерегистрирует Store
Шаг 5: Очищает кэш
Шаг 6: Настраивает Windows Update

========================================
  ПОСЛЕ ВЫПОЛНЕНИЯ:
========================================

1. Перезагрузите компьютер (ОБЯЗАТЕЛЬНО!)
2. Попробуйте открыть Microsoft Store

Если все еще не работает, выполните:

Get-AppXPackage *WindowsStore* | Reset-AppxPackage

Затем снова перезагрузите компьютер.

========================================
  АЛЬТЕРНАТИВНЫЕ КОМАНДЫ:
========================================

Если скрипт не помог, попробуйте вручную:

1. Сброс Store:
   Get-AppXPackage *WindowsStore* | Reset-AppxPackage

2. Очистка кэша:
   wsreset.exe

3. Перерегистрация:
   Get-AppxPackage -Name Microsoft.WindowsStore | Remove-AppxPackage
   Get-AppxProvisionedPackage -Online | Where-Object {$_.DisplayName -like "*WindowsStore*"} | ForEach-Object {Add-AppxProvisionedPackage -Online -PackagePath $_.PackagePath}
