========================================
НАЧНИТЕ ОТСЮДА - ВОССТАНОВЛЕНИЕ GRUB
========================================

У ВАС ЕСТЬ ФЛЕШКА - ОТЛИЧНО!

ПЛАН ДЕЙСТВИЙ:
--------------

1. СКАЧАТЬ LINUX MINT ISO
   https://linuxmint.com/download.php
   → Linux Mint 21.3 Cinnamon (64-bit)
   → Скачать торрент или прямой файл

2. СОЗДАТЬ ЗАГРУЗОЧНУЮ ФЛЕШКУ
   - Скачать Rufus: https://rufus.ie/
   - Запустить Rufus
   - Выбрать флешку
   - Выбрать ISO файл
   - Нажать "СТАРТ"
   - Дождаться завершения

3. ЗАГРУЗИТЬСЯ С ФЛЕШКИ
   - Вставить флешку
   - Перезагрузить
   - Нажать F12 (или F2, F8, ESC)
   - Выбрать USB в меню

4. ВОССТАНОВИТЬ GRUB
   - Открыть терминал (Ctrl+Alt+T)
   - Выполнить команды из файла: КОМАНДЫ_GRUB.txt
   - Или следовать инструкции: ВОССТАНОВИТЬ_GRUB_ШАГИ.txt

5. ПЕРЕЗАГРУЗИТЬ
   - Вынуть флешку
   - Должен появиться GRUB с выбором Mint/Windows

ПОДРОБНЫЕ ИНСТРУКЦИИ:
---------------------
→ ВОССТАНОВИТЬ_GRUB_ШАГИ.txt (полная инструкция)
→ КОМАНДЫ_GRUB.txt (готовые команды для копирования)

ВРЕМЯ: ~30 минут
СЛОЖНОСТЬ: Средняя (нужно скопировать команды)
