========================================
ВОССТАНОВЛЕНИЕ ЗАГРУЗЧИКА LINUX MINT
========================================

ПРОБЛЕМА: При загрузке сразу открывается Windows, нет выбора Mint

РЕШЕНИЕ 1: ЧЕРЕЗ LIVE USB (РЕКОМЕНДУЕТСЯ - 100% РАБОТАЕТ)
----------------------------------------------------------

ШАГ 1: Создать загрузочную флешку
----------------------------------
1. Скачайте Linux Mint ISO:
   https://linuxmint.com/download.php
   (выберите версию Cinnamon 64-bit)

2. Скачайте Rufus:
   https://rufus.ie/

3. Запустите Rufus:
   - Устройство: выберите вашу USB флешку
   - Образ: выберите скачанный ISO файл
   - Схема раздела: GPT (для UEFI) или MBR (для Legacy)
   - Нажмите "Старт"
   - Выберите "Записать в режиме ISO"

ШАГ 2: Загрузиться с флешки
----------------------------
1. Вставьте флешку в компьютер
2. Перезагрузите компьютер
3. При загрузке нажмите клавишу выбора загрузки:
   - F12 (Dell, HP, большинство компьютеров)
   - F2 (Acer, Asus)
   - F8 (Lenovo)
   - ESC (некоторые ноутбуки)
   - F10 (некоторые системы)
4. В меню загрузки выберите USB (обычно "USB HDD" или название флешки)

ШАГ 3: Восстановить GRUB
-------------------------
1. В Live Mint откройте терминал:
   Нажмите: Ctrl + Alt + T

2. Найдите раздел с вашим Mint:
   sudo fdisk -l
   
   Ищите раздел с типом "Linux" (обычно /dev/sda2, /dev/sda3, или /dev/nvme0n1p2)
   Запомните номер раздела!

3. Выполните команды (ЗАМЕНИТЕ /dev/sda2 на ваш раздел!):
   
   # Монтировать корневой раздел Mint
   sudo mount /dev/sda2 /mnt
   
   # Если есть отдельный EFI раздел (обычно /dev/sda1, размер ~100-500MB):
   sudo mount /dev/sda1 /mnt/boot/efi
   
   # Монтировать системные разделы
   sudo mount --bind /dev /mnt/dev
   sudo mount --bind /proc /mnt/proc
   sudo mount --bind /sys /mnt/sys
   sudo mount --bind /run /mnt/run
   
   # Перейти в систему Mint
   sudo chroot /mnt
   
   # Переустановить GRUB
   grub-install /dev/sda
   update-grub
   
   # Выйти из chroot
   exit
   
   # Размонтировать
   sudo umount /mnt/run
   sudo umount /mnt/sys
   sudo umount /mnt/proc
   sudo umount /mnt/dev
   sudo umount /mnt/boot/efi
   sudo umount /mnt
   
   # Перезагрузить
   sudo reboot

4. Выньте флешку при перезагрузке
5. Должен появиться GRUB с выбором Windows и Mint!

РЕШЕНИЕ 2: EASYBCD (БЕЗ LIVE USB - ПРОЩЕ)
------------------------------------------

1. Скачайте EasyBCD:
   https://neosmart.net/EasyBCD/
   (бесплатная версия)

2. Установите EasyBCD
   (может потребоваться .NET Framework)

3. Запустите EasyBCD от имени администратора:
   Правой кнопкой → Запуск от имени администратора

4. Добавьте Linux:
   - Нажмите "Add New Entry"
   - Выберите вкладку "Linux/BSD"
   - Type: выберите "GRUB 2"
   - Name: введите "Linux Mint"
   - Drive: выберите раздел с Mint
     (обычно это большой раздел без буквы, ~100-500GB)
   - Нажмите "Add Entry"

5. Перезагрузите компьютер
6. При загрузке выберите "Linux Mint" в меню

РЕШЕНИЕ 3: ИЗ WINDOWS (ПРОДВИНУТЫЙ)
------------------------------------

Запустите скрипт: fix-mint-bootloader.ps1
(от имени администратора)

Он покажет информацию о разделах и даст инструкции.

ВАЖНО:
------
- Перед любыми действиями сохраните важные данные
- Если не уверены в разделе - используйте Live USB метод
- После восстановления GRUB будет работать нормально

ПОМОЩЬ:
-------
Если что-то не работает:
1. Проверьте, что флешка загружается (попробуйте на другом компьютере)
2. Убедитесь, что выбрали правильный раздел с Mint
3. Если Mint не загружается - возможно проблема в самом Mint, не в загрузчике
