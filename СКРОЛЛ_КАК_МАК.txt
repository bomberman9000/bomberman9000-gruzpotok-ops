========================================
НАСТРОЙКА СКРОЛЛА КАК НА MACBOOK
========================================

ЧТО СДЕЛАНО:
------------
✓ Плавный скролл Windows включен
✓ Настройки мыши оптимизированы
✓ Firefox настроен для плавного скролла
✓ Браузеры настроены

ПРИМЕНЕНИЕ ИЗМЕНЕНИЙ:
---------------------

1. ПЕРЕЗАГРУЗИТЕ компьютер (лучший вариант)
   Или выйдите и войдите в систему

2. Если не хотите перезагружать:
   - Закройте все браузеры
   - Откройте их снова
   - Перезапустите другие приложения

НАСТРОЙКА FIREFOX (если нужно):
--------------------------------

1. Откройте Firefox
2. В адресной строке введите: about:config
3. Нажмите "Принять риск и продолжить"
4. Найдите и установите:

   general.smoothScroll = true
   general.smoothScroll.mouseWheel = true
   general.smoothScroll.pages = true
   mousewheel.min_line_scroll_amount = 5
   mousewheel.acceleration.start = 1
   mousewheel.acceleration.factor = 10
   toolkit.scrollbox.smoothScroll = true

5. Перезапустите Firefox

НАСТРОЙКА СКОРОСТИ СКРОЛЛА:
----------------------------

Если скролл слишком быстрый/медленный:

1. Параметры Windows → Устройства → Мышь
2. Настройте ползунок "Прокручивать неактивные окна при наведении"
3. Или измените "Строк за раз" (3 = стандарт)

Через PowerShell:
   $key = "HKCU:\Control Panel\Mouse"
   Set-ItemProperty -Path $key -Name "WheelScrollLines" -Value 5
   # 1-100: меньше = медленнее, больше = быстрее

ПРОВЕРКА:
---------

1. Откройте любой длинный сайт (например, новости)
2. Прокрутите колесиком мыши
3. Скролл должен быть:
   - Плавным (без рывков)
   - Быстрым (как на Mac)
   - Отзывчивым

ЕСЛИ НЕ РАБОТАЕТ:
-----------------

1. Убедитесь, что перезагрузили компьютер
2. Проверьте настройки мыши в Windows
3. Для Firefox используйте about:config (см. выше)
4. Попробуйте другой браузер (Chrome/Edge)

ДОПОЛНИТЕЛЬНО:
--------------

Для еще более плавного скролла в Firefox:
- Установите расширение "SmoothScroll"
- Или используйте настройки выше

Для Chrome/Edge:
- Настройки → Дополнительные → Система
- Включите "Использовать аппаратное ускорение"
