========================================
КАСПЕРСКИЙ УДАЛИЛ CURSOR - БЫСТРОЕ РЕШЕНИЕ
========================================

✅ Cursor найден в системе, но Касперский может блокировать его работу.

ДЕЙСТВИЯ (5 минут):
-------------------

1. ОТКРЫТЬ КАСПЕРСКИЙ
   - Правой кнопкой на иконку в трее → Открыть
   - Или: Пуск → Kaspersky

2. ДОБАВИТЬ ИСКЛЮЧЕНИЯ
   - Настройки (шестеренка) → Дополнительно
   - Угрозы и исключения → Настроить исключения
   - Добавить → Файлы и папки
   - Добавьте эти 3 пути:
   
     C:\Program Files\cursor\*
     C:\Users\Shata\AppData\Local\Programs\cursor\*
     C:\Users\Shata\project\*
   
   - Сохраните

3. ПРОВЕРИТЬ КАРАНТИН
   - Защита → Карантин
   - Найдите файлы cursor/project
   - Восстановите все найденные

4. ПЕРЕЗАГРУЗИТЬ КОМПЬЮТЕР
   - Это важно для применения исключений!

5. ЗАПУСТИТЬ CURSOR
   - После перезагрузки откройте Cursor
   - Должно работать!

ЕСЛИ НЕ ПОМОГЛО:
----------------
- Временно отключите Касперский (Защита → Отключить)
- Запустите Cursor
- Добавьте исключения
- Включите Касперский обратно
