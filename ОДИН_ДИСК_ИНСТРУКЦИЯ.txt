========================================
WINDOWS И MINT НА ОДНОМ ДИСКЕ - ВОССТАНОВЛЕНИЕ GRUB
========================================

ВАЖНО: Оба на одном физическом диске - это нормально!
GRUB восстановится и будет показывать оба варианта загрузки.

СТРУКТУРА ДИСКА (пример):
--------------------------
/dev/sda1  - EFI раздел (загрузчик, ~100-500MB)
/dev/sda2  - Linux Mint (корневой раздел, ~20-100GB)
/dev/sda3  - Linux swap (опционально, ~2-8GB)
/dev/sda4  - Windows (C:\, ~200-500GB)
/dev/sda5  - Windows Recovery

ШАГ 1: СОЗДАТЬ ЗАГРУЗОЧНУЮ ФЛЕШКУ
----------------------------------
1. Скачайте Linux Mint ISO: https://linuxmint.com/download.php
2. Скачайте Rufus: https://rufus.ie/
3. Запустите Rufus:
   - Устройство: ваша USB флешка
   - Образ: выберите ISO
   - Схема: GPT (для UEFI) или MBR (для старых систем)
   - СТАРТ → Дождитесь завершения

ШАГ 2: ЗАГРУЗИТЬСЯ С ФЛЕШКИ
----------------------------
1. Вставьте флешку
2. Перезагрузите компьютер
3. При загрузке нажмите F12 (или F2/F8/ESC)
4. Выберите USB в меню загрузки

ШАГ 3: НАЙТИ РАЗДЕЛЫ
--------------------
В терминале (Ctrl+Alt+T) выполните:

sudo fdisk -l

Вы увидите что-то вроде:
/dev/sda1   2048   1050623   524288   256M  EFI System
/dev/sda2  1050624 41943039 20446208   9.8G  Linux filesystem  ← ЭТО MINT!
/dev/sda3  41943040 46137343  2097152     1G  Linux swap
/dev/sda4  46137344 ...                Windows

Запомните:
- EFI раздел (обычно /dev/sda1)
- Раздел с Mint (обычно /dev/sda2)

ШАГ 4: ВОССТАНОВИТЬ GRUB
-------------------------
Выполните команды (ЗАМЕНИТЕ номера на ваши!):

# Монтировать корневой раздел Mint
sudo mount /dev/sda2 /mnt

# Монтировать EFI раздел (ВАЖНО для UEFI!)
sudo mount /dev/sda1 /mnt/boot/efi

# Монтировать системные разделы
sudo mount --bind /dev /mnt/dev
sudo mount --bind /proc /mnt/proc
sudo mount --bind /sys /mnt/sys
sudo mount --bind /run /mnt/run

# Войти в систему Mint
sudo chroot /mnt

# Установить GRUB на весь диск (это безопасно!)
grub-install /dev/sda

# Обновить конфигурацию GRUB (найдет Windows автоматически!)
update-grub

# Выйти
exit

# Размонтировать
sudo umount /mnt/run
sudo umount /mnt/sys
sudo umount /mnt/proc
sudo umount /mnt/dev
sudo umount /mnt/boot/efi
sudo umount /mnt

# Перезагрузить
sudo reboot

ШАГ 5: ПРОВЕРИТЬ
-----------------
1. Выньте флешку при перезагрузке
2. Должен появиться GRUB меню с:
   - Linux Mint (несколько вариантов)
   - Windows Boot Manager
3. Выберите нужную систему

ВАЖНО:
------
✓ GRUB НЕ повредит Windows - он только добавит меню загрузки
✓ Windows останется на своем разделе
✓ После восстановления сможете выбирать между Mint и Windows
✓ Если выберете Windows - загрузится как обычно

ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК:
--------------------------
- Windows все еще можно загрузить через BIOS/UEFI меню (F12)
- Можно восстановить Windows загрузчик через Windows Recovery
- GRUB можно переустановить снова
