========================================
ВОССТАНОВЛЕНИЕ ЗАГРУЗЧИКА LINUX MINT
========================================

ПРОБЛЕМА: Загрузчик GRUB не появляется, сразу загружается Windows

РЕШЕНИЕ 1: ВОССТАНОВЛЕНИЕ ЧЕРЕЗ LIVE USB (РЕКОМЕНДУЕТСЯ)
----------------------------------------------------------

1. Создайте загрузочную флешку Linux Mint:
   - Скачайте ISO: https://linuxmint.com/download.php
   - Используйте Rufus или Balena Etcher для записи на USB

2. Загрузитесь с USB:
   - Перезагрузите компьютер
   - Нажмите F12/F2/F8 (зависит от материнской платы)
   - Выберите USB в меню загрузки

3. В Live Mint откройте терминал и выполните:

   # Найти раздел с Mint (обычно /dev/sda2 или /dev/nvme0n1p2)
   sudo fdisk -l
   
   # Монтировать корневой раздел Mint
   sudo mount /dev/sda2 /mnt
   
   # Если есть отдельный раздел /boot/efi, монтировать его
   sudo mount /dev/sda1 /mnt/boot/efi
   
   # Монтировать системные разделы
   sudo mount --bind /dev /mnt/dev
   sudo mount --bind /proc /mnt/proc
   sudo mount --bind /sys /mnt/sys
   sudo mount --bind /run /mnt/run
   
   # Перейти в chroot
   sudo chroot /mnt
   
   # Переустановить GRUB
   grub-install /dev/sda
   update-grub
   
   # Выйти из chroot
   exit
   
   # Перезагрузить
   sudo reboot

РЕШЕНИЕ 2: ДОБАВИТЬ MINT В ЗАГРУЗЧИК WINDOWS (ВРЕМЕННОЕ)
----------------------------------------------------------

Если у вас UEFI система, можно добавить Mint в меню загрузки Windows:

1. Откройте PowerShell от имени администратора

2. Найдите раздел EFI с Mint:
   mountvol S: /S
   dir S:\EFI

3. Добавьте запись в BCD:
   bcdedit /copy {current} /d "Linux Mint"
   bcdedit /set {GUID} path \EFI\ubuntu\grubx64.efi
   (замените GUID на полученный из предыдущей команды)

РЕШЕНИЕ 3: ИСПОЛЬЗОВАТЬ EASYBCD (ПРОСТОЙ СПОСОБ)
--------------------------------------------------

1. Скачайте EasyBCD: https://neosmart.net/EasyBCD/

2. Установите и запустите от имени администратора

3. Добавьте Linux:
   - Add New Entry → Linux/BSD
   - Type: GRUB 2
   - Name: Linux Mint
   - Drive: выберите раздел с Mint
   - Add Entry

4. Перезагрузите компьютер
