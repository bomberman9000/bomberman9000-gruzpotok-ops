Шаблоны системных промптов для ботов (Ollama Modelfile: SYSTEM "...").

Файлы:
  assistant.txt  — универсальный помощник
  coder.txt      — разработка кода
  tutor.txt      — обучение / объяснения
  support.txt    — поддержка пользователей

Сборка своей модели из шаблона (пример):
  ollama create my-assistant -f ollama\Modelfile.assistant
