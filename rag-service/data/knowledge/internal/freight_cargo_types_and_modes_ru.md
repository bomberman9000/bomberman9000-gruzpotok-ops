# Грузы и способы перевозки (справочно для RAG)

Внутренний обзор для подсказок оператору и ответов по базе. Не заменяет ДОПОГ, правила перевозки конкретного груза, договор и инструкции перевозчика. Режимы и документы уточняйте по актуальным нормам и маршруту.

## 1. Что обычно понимают под «типом груза»

### По агрегатному состоянию

| Категория | Примеры | Заметки для логистики |
|-----------|---------|------------------------|
| Насыпные / навалочные | Зерно, щебень, уголь | Погрузка/выгрузка спец. оборудованием, учёт веса и влажности |
| Наливные | Нефтепродукты, химия в цистернах | Цистерны, совместимость материала ёмкости и продукта |
| Генеральные (штучные) | Тара, оборудование, паллеты | Упаковка, крепление, маркировка |
| Негабаритные / тяжеловесные | Оборудование, конструкции | Разрешения на перевозку, схемы, сопровождение по правилам |
| Скоропортящиеся | Продукты с коротким сроком | Рефрижераторы, температурный контроль, сроки |
| Опасные | Грузы по классам опасности (ДОПОГ и др.) | Класс/UN, упаковка, документы, допуски водителя/ТС |

### По температурному режиму

- **Изотерм / рефрижератор** — поддержание температуры по требованию груза и договора.
- **Без спецрежима** — стандартная сухая перевозка при допустимых для груза условиях.

## 2. Способы перевозки (виды транспорта)

| Способ | Когда чаще выбирают | Ограничения и нюансы |
|--------|---------------------|----------------------|
| **Автомобильный** | «Дверь—дверь», гибкость по маршруту и срокам | Пропускная способность дорог, весогабаритные рамки, тахограф, для международных автоперевозок — CMR при необходимости |
| **Железнодорожный** | Крупные объёмы, дальние расстояния по РФ/СНГ | Планирование подвижного состава и сроков, перегрузки при смене колеи/погранпереходах |
| **Морской** | Межконтинентальные потоки, крупнотоннаж | Сроки в пути, коносамент, стыковка с портовой и автомобильной доставкой |
| **Речной** | Внутренние водные пути при наличии линии | Сезонность, глубина, погрузочно-разгрузочные работы в портах |
| **Авиационный** | Срочность, высокая удельная стоимость доставки | Ограничения по опасным грузам и габаритам, аэропортовые процедуры |
| **Контейнерные перевозки** | Унификация перегрузки между видами транспорта | Тип контейнера (DC, HC, реф и т.д.), ответственность за состояние контейнера |
| **Мультимодальная / комбинированная** | Один договор — несколько этапов разным транспортом | Единый перевозочный документ при модели перевозчика, прослеживаемость этапов |

## 3. Связь «груз — способ» (эвристики, не правила)

- **Срочный мелкий высокоценный груз** — чаще авиа или экспресс-авто по согласованию.
- **Крупный объём на дальность** — ж/д или море + автодовоз.
- **Скоропортящийся** — реф, соблюдение цепочки холода, минимизация простоев.
- **Опасный** — только после проверки класса, упаковки, допусков и маршрута по ДОПОГ (или иным режимам).
- **Негабарит** — отдельное согласование маршрута, возможно сопровождение и спецтехника.

## 4. Что фиксировать в заявке и переписке

- Наименование груза, вес, объём, количество мест, габариты (длина × ширина × высота), тип упаковки.
- Требования по температуре, хрупкость, опасность (класс, UN при наличии).
- Адреса погрузки/выгрузки, желаемые окна, особенности доступа для ТС.
- Предпочтительный способ перевозки или «на усмотрение перевозчика» при известных ограничениях.
