# Основы подбора техники под груз (автоперевозки, справочно для RAG)

Внутренний конспект для операторов и ответов по базе. **Не заменяет** ПДД, допустимые массы/габариты ТС, ДОПОГ, технические паспорта конкретных машин, договор и инструкции перевозчика. Всегда сверяйте с фактическими характеристиками предлагаемого ТС и нормами на дату перевозки.

См. также: [freight_cargo_types_and_modes_ru.md](freight_cargo_types_and_modes_ru.md) (виды грузов и транспорта).

---

## 1. Какие исходные данные нужны до подбора техники

Минимум, без которого подбор остаётся «на глаз»:

| Параметр | Зачем |
|----------|--------|
| **Масса брутто** (кг/т) | Не превышать грузоподъёмность кузова и допустимые нагрузки на ось/сцепку по закону и разрешениям |
| **Объём** (м³) | Соответствие полезному объёму кузова, учёт коэффициента заполнения (паллеты не «жидк») |
| **Габариты** (Д × Ш × В) | Вход в кузов, погрузочные ворота, высота трубы, негабарит |
| **Количество мест / упаковка** | Паллеты, мешки, россыпь — влияет на погрузку и крепление |
| **Температурный режим** | От +… до +… °C или «не ниже» — реф, изотерм или без |
| **Совместимость груза** | Опасный груз (класс, UN), пищевой, гигроскопичный, требует изоляции от других грузов |
| **Особенности площадок** | Наличие рампы, крана, ворот, ограничения по высоте и весу на территории |

## 2. Типы кузовов и прицепов (автомобильные перевозки)

Ниже — **типовые назначения**, не жёсткий закон: одна и та же категория груза может ехать на разных типах ТС при соблюдении массы, габаритов и режима.

| Тип техники | Что обычно везут | Особенности |
|-------------|------------------|-------------|
| **Тент (штора)** | Паллеты, коробки, непогодочувствительный в надёжной упаковке | Защита от осадков, боковая/задняя погрузка; контроль намокания упаковки |
| **Бортовой** | Стройматериалы, оборудование, негабарит в пределах борта | Крепление ремнями/тросами, риск смещения при плохом креплении |
| **Изотермический фургон** | Продукты, косметика, лекарства при умеренном температурном режиме | Снижение перепадов температуры; не путать с «рефом» без холодильной установки |
| **Рефрижератор (реф)** | Скоропорт, заморозка, цепочка холода | Температура по договору, записи/простыва при необходимости, проверка исправности установки |
| **Рефрижератор мультитемпературный** | Разные зоны на одном рейсе | Зонирование и режимы по секциям |
| **Контейнеровоз (полуприцеп-платформа)** | 20/40/45 футовые контейнеры | Соответствие типа контейнера (DC, HC, реф-контейнер) и крепежа twist-lock |
| **Самосвал / кузов с подъёмом** | Сыпучие, стройматериалы (по согласованию) | Не смешивать с типичной паллетной логистикой без отдельного ТЗ |
| **Цистерна (бензовоз, химия в цистерне)** | Наливные нефтепродукты, химия в допустимых для цистерны классах | Совместимость материала ёмкости и продукта, обучение водителя, ДОПОГ при необходимости |
| **Низкорамный трал / низкорамник** | Высокое оборудование, станки, негабарит в пределах разрешения | Часто разрешения сопровождения и схема маршрута |
| **Платформа с тяжёлым креплением** | Крупногабаритные грузы по согласованию | Проектная схема крепления, возможно сопровождение |

**Автопоезд** (тягач + полуприцеп или сцепка) увеличивает допустимую массу по сравнению с одиночным грузовиком в рамках норм, но требует проверки по типу сцепки и дорогам.

## 3. Логика «груз — техника» (по шагам)

1. **Ограничить по массе и объёму** — груз не должен превышать полезную загрузку и объём кузова с запасом на упаковку и крепёж.
2. **Проверить габариты** — длина/ширина/высота груза меньше внутренних размеров кузова (или план негабарита с разрешениями).
3. **Температура** — если задан диапазон °C → реф или изотерм по требованию груза и договора.
4. **Опасный груз** — только после класса, UN, упаковки, допусков водителя и ТС по ДОПОГ (или иным правилам); тип цистерны/кузова по регламенту.
5. **Наливной** — цистерна соответствующего назначения, не «общий тент».
6. **Контейнерная перевозка** — контейнеровоз + контейнер нужного типа; для реф-контейнера — питание/режим по согласованию.
7. **Негабарит** — отдельный расчёт маршрута, разрешения, возможно сопровождение и спецтехника погрузки.

---

## 4. Плотность и «вес против объёма»

- Лёгкий объёмный груз (пенопласт, пустая тара) может **упереться в объём** кузова раньше, чем в массу.
- Тяжёлый компактный груз (металл, плитка) — чаще **упирается в массу** и осями.
- Насыпные грузы — учёт **угол естественного откоса**, влажности; объём не равен «литражу» вагона без коэффициента.

---

## 5. Крепление груза (базовые принципы)

- Груз не должен **смещаться** при торможении и поворотах — стяжные ремни, противоскольжение, обрешётка по необходимости.
- Тяжёлые предметы — **ниже центр массы**, равномерное распределение по осям в рамках паспорта ТС.
- Острые кромки — **защита** ремней и упаковки.
- Для опасных и специальных грузов — **нормативные схемы** крепления и маркировка.

---

## 6. Типичные ошибки подбора

| Ошибка | Последствия |
|--------|----------------|
| Реф заказан без уточнения температуры | Порча груза, споры |
| Тент без учёта влажности/гигроскопичности | Повреждение упаковки |
| Перегруз по массе или осям | Штрафы, повреждение дороги и ТС, отказ в перевозке |
| Негабарит без разрешений | Простой, штрафы, возврат |
| Несовместимые грузы в одном кузове | Загрязнение, порча, отказ от ответственности |
| Цистерна не по назначению продукта | Аварийный риск, несоответствие нормам |

---

## 7. Что передавать в заявке перевозчику для точного подбора

- Название груза, вес, объём, габариты, число мест.
- Температурный режим или «нет».
- Опасность: класс, UN, при необходимости паспорт безопасности.
- Адреса погрузки/выгрузки, желаемые окна, ограничения по высоте ворот и весу на территории.
- Нужен ли **контейнер**, **цистерна**, **трал**, **только реф** — или допускается «аналог при равных условиях».

---

Документ можно дополнять **внутренними стандартами компании** (минимальный запас по массе, список одобренных типов ТС по категориям грузов), не включая персональные данные контрагентов.
