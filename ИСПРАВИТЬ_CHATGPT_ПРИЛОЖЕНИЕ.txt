========================================
ЕСЛИ НЕ РАБОТАЕТ:
========================================

ПРОБЛЕМА 1: Приложение не запускается
--------------------------------------

Решение:
1. Проверьте, что приложение установлено:
   Get-AppxPackage | Where-Object { $_.Name -like "*chatgpt*" }

2. Переустановите приложение из Microsoft Store

3. Запустите от имени администратора

ПРОБЛЕМА 2: Приложение запускается, но не работает
---------------------------------------------------

Решение:
1. Проверьте VPN:
   .\check-vpn.ps1

2. Убедитесь, что Hysteria2 работает:
   Get-Process -Name "hysteria2"

3. Проверьте WinHTTP прокси:
   netsh winhttp show proxy
   
   Должно показать: 127.0.0.1:8080

4. Если не настроен, настройте:
   netsh winhttp set proxy proxy-server="127.0.0.1:8080" bypass-list="<local>"

ПРОБЛЕМА 3: Ошибка подключения
-------------------------------

Решение:
1. Перезапустите Hysteria2:
   Stop-Process -Name "hysteria2" -Force
   .\start-hysteria2.ps1

2. Перезапустите приложение ChatGPT

3. Проверьте порты:
   netstat -an | findstr "8080 1080"
   
   Должны быть в состоянии LISTENING

ПРОБЛЕМА 4: Все еще не работает
--------------------------------

Решение:
1. ПЕРЕЗАГРУЗИТЕ компьютер
   - Это применит все настройки WinHTTP
   - UWP приложения перезапустятся с новыми настройками

2. После перезагрузки:
   - Запустите: .\start-hysteria2.ps1
   - Запустите: .\launch-chatgpt.ps1

3. Проверьте VPN:
   .\check-vpn.ps1

========================================
ПРОВЕРКА НАСТРОЕК:
========================================

1. Проверить Hysteria2:
   Get-Process -Name "hysteria2"

2. Проверить порты:
   netstat -an | findstr "8080 1080"

3. Проверить системный прокси:
   Get-ItemProperty "HKCU:\Software\Microsoft\Windows\CurrentVersion\Internet Settings" | Select-Object ProxyEnable, ProxyServer

4. Проверить WinHTTP прокси:
   netsh winhttp show proxy

5. Проверить переменные окружения:
   $env:HTTP_PROXY
   $env:HTTPS_PROXY

========================================
КОМАНДЫ ДЛЯ ТЕРМИНАЛА:
========================================

Запустить ChatGPT:
  .\launch-chatgpt.ps1

Проверить VPN:
  .\check-vpn.ps1

Запустить Hysteria2:
  .\start-hysteria2.ps1

Остановить Hysteria2:
  Stop-Process -Name "hysteria2" -Force

  netsh winhttp set proxy proxy-server="127.0.0.1:8080" bypass-list="<local>"

Проверить WinHTTP прокси:
  netsh winhttp show proxy

Сбросить WinHTTP прокси:
  netsh winhttp reset proxy
