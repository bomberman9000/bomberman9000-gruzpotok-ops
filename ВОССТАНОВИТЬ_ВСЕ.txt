СТАТУС:
--------
✓ Cursor установлен в: C:\Program Files\cursor
✗ Локальные файлы могут быть удалены

ШАГ 1: ПРОВЕРИТЬ КАРАНТИН КАСПЕРСКОГО
-------------------------------------
1. Откройте Касперский
2. Защита → Карантин
3. Найдите все файлы связанные с:
   - cursor
   - Cursor
   - project (ваши скрипты)
4. Для каждого файла: Правой кнопкой → Восстановить

ШАГ 2: ДОБАВИТЬ ИСКЛЮЧЕНИЯ (ОБЯЗАТЕЛЬНО!)
-----------------------------------------
Откройте файл: КАСПЕРСКИЙ_ИСКЛЮЧЕНИЯ.txt
Следуйте инструкциям там.

Или быстро:
1. Касперский → Настройки → Дополнительно → Угрозы и исключения
2. Настроить исключения → Добавить
3. Добавьте:
   ✓ C:\Program Files\cursor\*
   ✓ C:\Users\Shata\AppData\Local\Programs\cursor\*
   ✓ C:\Users\Shata\project\*

ШАГ 3: ПРОВЕРИТЬ CURSOR
------------------------
1. Запустите Cursor из меню Пуск
2. Если не запускается - переустановите с https://cursor.sh
3. Откройте проект: C:\Users\Shata\project

ШАГ 4: ВОССТАНОВИТЬ ПРОЕКТ
---------------------------
Если файлы проекта удалены:
1. Проверьте карантин Касперского
2. Восстановите все .ps1 и .bat файлы
3. Если не найдены - используйте Git (если был) или восстановите из резервной копии

ВАЖНО:
------
После добавления исключений перезагрузите компьютер,
чтобы Касперский применил новые правила!
